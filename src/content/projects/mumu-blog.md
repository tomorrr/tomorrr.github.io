从头搭建的个人博客，记录技术探索和生活随想。

### 技术亮点
- 纯 CSS 玻璃拟态效果，无第三方 UI 库
- 星空粒子 Canvas 背景动画
- React 岛屿架构，只在交互处加载 JS
- Framer Motion 动效
- 全静态输出，GitHub Pages 一键部署
