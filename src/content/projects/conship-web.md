一个配置文件托管平台，支持在线管理和分享配置文件。适合团队协作和配置备份场景。

## 功能特性

- 用户注册与登录
- 配置文件上传、编辑、删除
- 公开 / 私有分享设置
- 版本历史记录
- 在线预览配置文件内容

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Vue 2 |
| UI 组件 | Element UI |
| 状态管理 | Vuex |
| 路由 | Vue Router |
| 构建工具 | Vue CLI |

## 本地运行

```bash
git clone https://github.com/tomorrr/Conship-web.git
cd Conship-web
npm install
npm run serve
```
