一份系统整理的前端面试复习资料，涵盖从基础到进阶的核心知识点。

## 内容概览

### HTML & CSS
- 语义化标签与 SEO
- Flexbox 与 Grid 布局
- BFC、层叠上下文
- 响应式设计原理

### JavaScript
- 原型链与继承
- 闭包、作用域、this
- 事件循环 (Event Loop)
- Promise、async/await
- 深拷贝与浅拷贝

### 框架 & 工具
- React 核心概念（虚拟 DOM、Hooks、状态管理）
- Vue 响应式原理
- Webpack / Vite 构建工具
- 前端性能优化

## 使用方式

```bash
git clone https://github.com/tomorrr/Front-End-Interview-Notebook.git
```

按目录浏览各模块笔记，建议结合 MDN 文档和实战项目一起学习。
