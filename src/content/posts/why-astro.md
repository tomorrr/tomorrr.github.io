## 选型的纠结

搭博客之前，我把主流的静态站点方案都试了一遍。

### Next.js
太重型了。一个博客而已，不需要服务端渲染也不需要 API 路由。打包出来几百 KB 的 JS，对读者很不友好。

### Hugo
速度确实快，模板语法也够用。但要做玻璃拟态这种视觉效果，Hugo 的模板系统让我头疼。

### Astro
刚刚好。默认零 JS 输出，内容和样式直接编译成 HTML。需要交互的地方嵌入 React 组件，不需要的地方就是纯静态。这就是我要的。
