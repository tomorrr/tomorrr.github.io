# 学习笔记

> 个人学习笔记仓库，记录日常技术学习的知识点、代码片段和踩坑经验。

## 笔记目录

- **JavaScript 深入** — 原型、闭包、异步编程、ES6+
- **TypeScript** — 类型系统、泛型、工具类型
- **Node.js** — 模块系统、Express、中间件
- **数据库** — MySQL 基础、Redis 缓存
- **开发工具** — Git 工作流、VS Code 配置、Linux 常用命令
- **算法与数据结构** — 排序、树、动态规划入门

## 笔记风格

- 每个知识点配有代码示例
- 重点标注常见陷阱和最佳实践
- 结合实际项目经验总结

## 使用方式

```bash
git clone https://github.com/tomorrr/Note.git
```

按目录浏览，推荐使用支持 Markdown 的编辑器（如 VS Code、Typora）阅读。
